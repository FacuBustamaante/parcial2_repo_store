## Instalación y uso

### 1. Clonar el repositorio

```bash
git clone https://github.com/FacuBustamaante/parcial2_repo_store.git
cd parcial2_repo_store
```

### 2. Instalar dependencias

```bash
pnpm install
```

### 3. Levantar el servidor de desarrollo

```bash
pnpm run dev
```
